Применение ООП в JavaScript

Создан класс GridView который имеет метод render(). Метод способен строить таблицы на основе передаваемых данных в виде двумерного массива arr = [ { vendorCode: '21006', name: 'Циркулярная пила', price: '5 600 ₽', stock: '7', sales: '23', reviews: '21',}, {}, ]. Вывод количества столбцов настраивается с помощью передаваемых атрибутов в виде массива.
attribute: { 'vendorCode': { 'label': 'Артикул', }, 'name': { 'label': 'Наименование', }, 'price': { 'label': 'Цена', }, },

От класса GridView наследуется новый класс GridViewSort который расширяет базовый класс, добавляя метод для сортировки столбцов.
